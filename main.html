Question-1: What are the different ways to select an element in the DOM?

getElementById(). getElementsByTagName(). getElementsByClassName(). querySelector() .querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML → Used to get or modify content along with HTML. innerText → Retrieves only the visible text. textContent → Ignores HTML and gets the full text, even if hidden

Question-3: What is event delegation in the DOM?

Event Delegation is a technique where an event listener is added to a parent element to handle events on its child elements. It works using Event Bubbling and helps improve performance, memory efficiency

Question-4: What is event bubbling in the DOM?

Event Bubbling is a process where an event propagates from the child element to the parent element step by step. It is enabled by default and can be stopped using event.stopPropagation()

Question-5: How do you create, add, and remove elements using JavaScript?

createElement() → Creates a new element. appendChild() → Adds an element. removeChild() → Removes an element
